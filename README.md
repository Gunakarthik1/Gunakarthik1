<br>

```
Discover → Enrich → Score (1-10) → Tailor Resume → Cover Letter → Auto-Apply
```

- Hits **8+ job boards**, filters for entry-level / OPT-eligible roles

- Vision model fills forms — handles CAPTCHAs, SSO gates, email walls
