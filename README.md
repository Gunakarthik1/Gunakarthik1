### 🤖 HireAgent — AI Job Application Agent

<details>
<summary><b>Click to expand details</b></summary>

> An end-to-end AI agent that automates the entire job application pipeline — from discovery to submission.

**6-Stage Pipeline:**
```
[Discover] → [Enrich] → [Score 1-10] → [Tailor Resume] → [Cover Letter] → [Auto-Apply]
```

- Scrapes **8+ job boards** for entry-level / OPT-eligible roles
- Uses **NVIDIA DeepSeek-V3** for scoring and **DeepSeek-R1** for resume tailoring
- **Vision-verified** form filling via Set-of-Marks + Llama-3.2-Vision
- Solves **hCaptcha, reCAPTCHA v2/Enterprise, Cloudflare Turnstile, Arkose Labs**
- Real-time **Telegram notifications** on application status

</details>

### 🚗 VinSight — Vehicle Valuation Engine

<details>
<summary><b>Click to expand details</b></summary>

> Instant, key-free vehicle appraisal tool — no signup, no API keys needed.

- Decodes any **17-digit VIN** via NHTSA VPIC
- Market value = `Base MSRP × age × mileage × condition × regional adjustment`
- Pulls **live recall data** from NHTSA
- **12-month depreciation projections** plotted with Chart.js
- **Hold/Sell recommendations** based on vehicle metrics
- "My Garage" with localStorage persistence

</details>
